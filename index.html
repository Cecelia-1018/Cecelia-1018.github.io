Supermama Social Media App

The current generation mother having the issues of too much stress, need second financial support, limited time and fatigue and work-life challenge that need them to face.This project have objectives on resolve those problem with create social posts on feeds or entertainment types, create chat room features, business modules, provide open forum for post question in anonymous mode, allow filter search and bookmark. React Native with Firebase to build mobile app whereas React With Firebase use to build the admin web app. The testing collect from around 35 or more than that overall show positive feedbacks. The project did support mother to resolve current problem and develop and launch out for potential clients to use.This app is created by Wong Wai Yi and Cecelia Lim.
