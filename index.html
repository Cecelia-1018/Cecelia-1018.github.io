2021 Projects

Employee Database Management

Employee Management System on AWS enables users to store employee information in a database, obtain employee data from a database as needed, edit employee information, delete employee information, take employee attendance, and examine the employee's attendance record. Following services that used to achieve the requirements of the small company. First, the Amazon Elastic Compute Cloud (EC2) is an IaaS that used to choose an OS to host the system, such as Amazon Linux 2. The static assets, such as employee profile photographs, are stored in Amazon Simple Storage Service (Amazon S3). Additionally, Amazon Relational Database Service (Amazon RDS) is a managed SQL database service that used with the database engine MySQL to store employee and attendance table records. AWS CloudWatch is also used to track the monitoring and observation of such AWS resources. Amazon Auto Scaling assists in monitoring the performance of compute resources by maintaining or increasing their size. Finally, the Amazon Elastic Load Balancing is used in the allocation of the incoming traffic on the application across the running instances.
